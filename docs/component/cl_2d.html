Chiu-Lin sharpening (2D)

RHS for the volume fraction \( \psi \):

\[ \text{RHS} = \nabla \cdot \left( \epsilon \nabla\psi - \psi(1-\psi) \hat{n} \right) \]

where the unit normal is:

\[ \hat{n} = \frac{\nabla\psi}{|\nabla\psi|} \]

In 2D, the flux vector is:

\[ \mathbf{q} = \left( \epsilon \frac{\partial\psi}{\partial x} - \psi(1-\psi) n_x, \; \epsilon \frac{\partial\psi}{\partial y} - \psi(1-\psi) n_y \right) \]

and the RHS is the divergence:

\[ \text{RHS} = \frac{\partial q_x}{\partial x} + \frac{\partial q_y}{\partial y} \]

Update:

\[ \psi^{n+1} = \psi^n + \Delta t \cdot \Gamma \cdot \text{RHS} \]

Parameters:

  • eps_target: Target interface thickness \( \epsilon \)
  • strength: Sharpening strength \( \Gamma \)

Note: This method is automatically selected when a 2D domain is specified and sharpening.method: cl is used.
