intsharp/monitors/gif.py

Animated MP4 Monitor

Creates MP4 videos from field snapshots. Identical functionality to the gif monitor but outputs MP4 format instead.

Requires: ffmpeg installed on your system.

Parameters

Same as gif monitor.

Output

  • Single-field: {field}.mp4
  • Multi-field: compare.mp4

Example

monitors:
  - type: mp4
    every_n_steps: 10
    field: alpha
monitors:
  - type: mp4
    every_n_steps: 10
    compare_fields:
      - field: alpha_no_sharp
        color: blue
      - field: alpha_pm
        color: red
