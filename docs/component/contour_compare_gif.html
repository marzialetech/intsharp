Contour Compare GIF Monitor (2D only)

Creates an animated GIF comparing contour lines of multiple 2D fields on the same plot. Each field can have its own contour levels, color, and line style.

YAML Configuration

monitors:
  - type: contour_compare_gif
    every_n_steps: 20
    compare_fields:
      - field: alpha_no_sharp
        contour_levels: [0.5]       # default [0.5]
        color: blue                  # auto-assigned if omitted
        linestyle: "-"               # default "-"
      - field: alpha_pm
        contour_levels: [0.25, 0.5, 0.75]
        color: red
        linestyle: "--"

Use with per-field sharpening

Combine with per-field sharpening control to compare sharpened vs non-sharpened fields:

fields:
  - name: alpha_no_sharp
    sharpening: false
  - name: alpha_pm
    sharpening: true
