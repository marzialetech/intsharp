intsharp/solvers/upwind.py

First-order upwind advection (2D)

Solves the 2D linear advection equation:

\[ \frac{\partial f}{\partial t} + u \frac{\partial f}{\partial x} + v \frac{\partial f}{\partial y} = 0 \]

Uses dimension-by-dimension operator splitting:

\[ f^* = f^n - \text{CFL}_x \cdot (f^n - f^n_{\text{upwind-x}}) \]
\[ f^{n+1} = f^* - \text{CFL}_y \cdot (f^* - f^*_{\text{upwind-y}}) \]

where \( \text{CFL}_x = u \, \Delta t / \Delta x \) and \( \text{CFL}_y = v \, \Delta t / \Delta y \).

For periodic boundary conditions, upwinding uses np.roll along each axis.

Stability: Requires \( \max(\text{CFL}_x, \text{CFL}_y) \leq 1 \).

Note: This solver is automatically selected when a 2D domain is specified and solver.type: upwind is used.
