intsharp/sharpening.py

Parameswaran-Mandal sharpening (2D)

RHS for the volume fraction \( \psi \):

\[ \text{RHS} = -K \psi(1-\psi)(1-2\psi) + \epsilon (1-2\psi) |\nabla\psi| \]

where \( K = 1/(4\epsilon^2) \) and the gradient magnitude is:

\[ |\nabla\psi| = \sqrt{\left(\frac{\partial\psi}{\partial x}\right)^2 + \left(\frac{\partial\psi}{\partial y}\right)^2} \]

Gradients are computed using central differences with periodic or one-sided boundary handling.

Update:

\[ \psi^{n+1} = \psi^n + \Delta t \cdot \Gamma \cdot \text{RHS} \]

Parameters:

  • eps_target: Target interface thickness \( \epsilon \)
  • strength: Sharpening strength \( \Gamma \)

Note: This method is automatically selected when a 2D domain is specified and sharpening.method: pm is used.
