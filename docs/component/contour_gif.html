Contour GIF Monitor (2D only)

Creates an animated GIF showing contour lines of a 2D field. Optionally displays the center of mass (centroid) and crosshairs through the centroid.

YAML Configuration

monitors:
  - type: contour_gif
    every_n_steps: 20
    field: alpha
    contour_level: 0.5      # default 0.5
    show_centroid: true     # show centroid marker
    show_crosshairs: true   # show crosshair lines through centroid
