Description

Short run exercising the txt and curve monitors. Output: columnar snapshot_*.txt and alpha_*.curve (x, value) files.

How to run

python run.py unit_tests/test_txt_curve.yaml

Before the first run, the YAML lives in unit_tests/; after a run it is moved into the timestamped run folder.

  • Time: dt = 0.05, 10 steps

  • Monitors: console, txt (every_n_steps: 5), curve (every_n_steps: 5)
