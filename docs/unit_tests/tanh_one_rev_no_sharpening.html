Description

Advection only, no sharpening. Tanh-hat profile, one revolution in a periodic domain. Output: 10-frame GIF.

How to run

python run.py unit_tests/tanh_one_rev_no_sharpening.yaml

The YAML lives in unit_tests/; each run copies it into the timestamped run folder (original remains for easy re-running).
