tanh_five_rev_cfl05_cl_sharpening

← Unit Tests

Description

Same setup as tanh_five_rev_cfl05_no_sharpening with Chiu-Lin (CL) sharpening enabled. Tanh-hat, 5 revolutions, CFL = 0.5.

How to run

python run.py unit_tests/tanh_five_rev_cfl05_cl_sharpening.yaml

The YAML lives in unit_tests/; each run copies it into the timestamped run folder (original remains for easy re-running).

  • Time: dt = 0.01, 1000 steps (T = 10 s, 5 revolutions)
  • CFL: 0.5
  • Velocity: 0.5
  • Field: alpha, IC tanh-hat, periodic BC
  • Solver: upwind; Timestepper: euler
  • Sharpening: CL, eps_target 0.02, strength 1.0
  • Monitors: console (every 200), gif (every 40 = 25 frames)
